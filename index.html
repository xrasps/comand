Установка и запуск React с Vite 1. npm create vite 2. projectName 3. Выбираем React 4. Выбираем JavaScript 5. cd projectName 6. npm install 7. npm run dev

В React имеется своя система маршрутизации, которая позволяет сопоставлять запросы к приложению с определенными компонентами. Ключевым звеном в работе маршрутизации является модуль react-router, который содержит основной функционал по работе с маршрутизацией. Однако если мы собираемся работать в браузере, то нам также надо использовать модуль react-router-dom Установка: npm install react-router-dom
